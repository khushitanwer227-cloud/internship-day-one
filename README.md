## Git Commands Used

## Steps Performed
Installed Visual Studio Code (VS Code) and Git on my system.
Created a project folder named Day-one.
Opened the project folder in VS Code.
Initialized a local Git repository using the git init command.
Created a README.md file inside the project folder.
Added project files to the staging area using git add ..
Created the first commit using git commit -m "Day 1 completed".
Created a new repository on GitHub.
Connected the local repository to the GitHub repository.
Pushed the project to GitHub using git push.
Verified that the repository was successfully uploaded to GitHub.
